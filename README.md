# text-cleaning-pipeline
A lightweight and efficient text preprocessing pipeline that cleans raw real-world text data and prepares it for NLP and machine learning applications.

## Features
- Converts text to lowercase
- Removes HTML tags
- Removes URLs
- Removes email addresses
- Normalizes extra spaces
- Removes punctuation and special characters
- Removes common stopwords
- Interactive user input

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run
1. git clone https://github.com/yourusername/text-cleaning-pipeline.git
2. cd text-cleaning-pipeline

*Run the program*

python main.py
